template<size_t Directions>
virtual void planner::cPlannerInterface< Directions >::TraversePath ( tNode *  i_psNode) const
pure virtual

Virtual abstract method of the base interface, which must be implemented to output the best found path.

The nodes in the best path contain a field tNode::psParent which makes it possible to move back to the start node, which has its parent pointer set to nullptr. This method is also used in planner::cPlanner::Plot() to output intermediate paths on an output image.

Parameters
[in]Thenode which on which we walk back. Usually the goal node.
