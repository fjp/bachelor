Detailed Description

The concrete cAudiRover class implementation of the interface cRoverInterface<size_t Direction>.

Provides eight actions and a summon feature.

The summon feature that the Audi rover provides.

Calls InitializePlanner which then acts as a factory to create a planner according to i_strAlgorithm parameter. Sets the step and velocity to 1. Note: step size and velocity should always be set to 1.

Parameters
i_strAlgorithmRvalue reference input specifies the algorithm to use. Possible strings "ASTAR", "ASTAR_WIKI", "ASTAR_RBG".
i_nStepSizethe step size of the rover. Should be set to 1.
i_nVelocitythe velocity of the rover. Deprecated. Should be set to 1.
Returns
Planning result struct with information such as travelling time and heuristic consistency.
